let.const

hoisting : 함수가 실행되기 전 변수들을 범위의 최상단으로 올리는것

  • const
    상수 (변하지 않는 값)
  • let
    1.ES6 자바스크립트 업그레이드 이후 나옴
    2. 지역변수로 사용하면 지역변수로 사용됨(var와 상이)
    3. 함수 선언문 보다 실행문이 상단에 있을경우
    let 호이스팅이 되지만, Temporal Death Zone(TDZ)으로 a 선언문이 나오기 전까지 에러 
                            console.log(a); 결과 = error
                        var a = 2;
                        console.log(a);
  • var
    1. ES6 자바스크립트 업그레이드 전
    2. 전역변수와 지역변수의 개념이 확실하지 않다.
    3. 동일 변수명도 실행이 가능 
                        var a =1;
                    console.log(a); 결과 = 1
                    var a = 2;
                    console.log(a); 결과 = 2
  • 전역변수
    블락 밖에서 선언을 한 변수(어디에서든 사용 가능)
  • 지역변수
    function이나, for문안에서 선언한 변수(블락 밖에서 사용이 불가능)
    단! var 는 지역변수로 사용해도, 함수를 제외하고 전역변수로 사용됨
