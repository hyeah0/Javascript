javacript [document.]은 html에 접근할 수 있다(읽을 수 있다.).

Test

        const text1 = document.getElementById("text1I");
        console.log(text1);
        console.dir(text1);
        
        console.log(text1.id);
        console.log(text1.className); 
        text1.innerText="change Text";
  1. const text1 = document.getElementById("text1I");
    text1 변수에 html id가 text1I을 할당한다
  2. console.log(text1);
    text1 값을 가져온다
    console.dir(text1);
    text1 객체(object)을 가져온다
  3. console.log(text1.id);
    text1의 id 명을 가져온다
    console.log(text1.className);
    text1의 class 명을 가져온다
  4. text1.innerText="change Text";
    text1 값의 text를 change text로 변경한다

        const text2 = document.getElementsByTagName("h1");
        console.log(text2);

        const text2_1 =document.querySelector(".text2");
        console.log(text2_1);

        const text2_2 = document.querySelector(".text2 h1");
        console.log(text2_2);
        
        const text2_3 = document.querySelectorAll(".text2 h1");
        console.log(text2_3);
  1. getElementsByTagName
    html 특정 태그를 가지고 오고 싶을때
  2. querySelector
    id를 가져올때는 "#id명", class를 가져올때는 ".class명"
    css처럼 부모요소 아래 하위 요소 가져올때 동일태그 자식이 많을 경우 첫번째만 확인이 가능하다
  3. querySelectorAll
    css처럼 부모요소 아래 하위 요소 가져올때 동일태그 자식이 많을 경우, 자식들 모두 확인하고 싶을때querySelectorAll사용한다
    특정 자식만 확인하고 싶을때 console.log(text2_3[2]);
